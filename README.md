## ⚙️ Installation & Setup  

### Clone the repo  
```bash
git clone https://github.com/GitSetGoRishabh/sih-2025-toursafe.git
cd sih-2025-toursafe
```

### Backend setup
```bash
cd backend
npm install   # or pip install -r requirements.txt
npm start     # or python manage.py runserver
```

### Frontend setup
```bash
cd frontend
npm install
npm run dev
```
